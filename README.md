# Random-Password-Generator
## Project Description 
This is a simple, robust, and highly customizable random password generator software. It allows users to quickly create secure passwords based on specific criteria, such as length, and 
the inclusion of various character types uppercase letters, lowercase letters, numbers, and symbols. The goal of this tool is to help users and developers generate strong, unique passwords to enhance their 
digital security.

## Key Features
* Generates strong random passwords
* Includes letters, numbers, and special characters
* Customizable password length
* Beginner-friendly and easy to modify

## How to use
* Install Python (if not installed already).
* Run the script using: python password:Random Passwoprd Generator.p
* Enter your desired password length.
* A secure password will be generated instantly.

## Example of Output
* Enter password length: 12
* Generated Password: a9#Kp3Z!wD1@
